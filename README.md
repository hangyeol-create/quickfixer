# QuickFixer

업무에서 자주 쓰는 파일 변환 도구 모음이에요.  
모든 처리는 **내 컴퓨터 안에서만** 이루어져요. 파일이 외부로 전송되지 않아요.

## 기능

- **Mac → Win 인코딩 변환** : Mac에서 만든 엑셀 파일을 Windows에서 깨지지 않게 변환
- **HWP → PDF 변환** : HWP 파일을 PDF로 변환

---

## 사용 방법

### 1단계. Node.js 설치 (처음 한 번만)

👉 [nodejs.org](https://nodejs.org) 에서 **LTS 버전** 다운로드 후 설치

### 2단계. 파일 받기

아래 중 편한 방법으로 받으세요.

**방법 A. ZIP으로 받기 (간단)**  
이 페이지 오른쪽 위 `Code` 버튼 → `Download ZIP` → 압축 해제

**방법 B. git으로 받기**
```
git clone https://github.com/hangyeol-create/quickfixer
```

### 3단계. 실행

`quickfixer` 폴더 안에 있는 **`start.bat`** 을 더블클릭하세요.

처음 실행할 때는 패키지 설치로 1~2분 걸릴 수 있어요.

### 4단계. 브라우저에서 열기

터미널에 `http://localhost:5173` 이 뜨면 브라우저에서 해당 주소로 접속하세요.  
(크롬 권장)

---

## 종료 방법

`start.bat` 실행 중인 터미널 창을 닫으면 돼요.
